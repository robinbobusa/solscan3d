# Solscan 3D

Локальный инструмент для исследования потоков средств в сети **Solana**: CLI к Solscan Pro API
плюс интерактивный **3D-граф** связей между кошельками (кто кому что отправлял), который
открывается в браузере. Весь обмен с API делает локальный Python-процесс — **токен остаётся на
сервере и в браузер не передаётся**.

## Что внутри

| Файл | Назначение |
|------|------------|
| `solscan.py` | CLI к Solscan Pro API: переводы, DeFi-активность, метаданные токена, разбор транзакций, мониторинг кошелька в реальном времени, экспорт в CSV. |
| `solscan_gui.py` | Локальный HTTP-сервер + графический интерфейс: вводишь адреса → строишь 3D-граф потоков. |
| `solscan_ui.html` | Сам интерфейс (3D-сцена, форма, список адресов). |
| `lib/` | Сторонние JS-библиотеки (three.js и др.) — подключаются локально, без CDN. |

## Требования

- **Python 3.8+**
- Пакет `requests` (`pip install -r requirements.txt`)
- **Бесплатный API-ключ Solscan Pro** — получить на <https://pro-api.solscan.io>

## Установка

```bash
git clone https://github.com/robinbobusa/solscan-3d.git
cd solscan-3d
pip install -r requirements.txt
```

## Настройка токена

Ключ задаётся **только** через переменную окружения `SOLSCAN_TOKEN` (в коде ключа нет — это сделано
намеренно, чтобы случайно не закоммитить секрет).

**Windows (PowerShell):**
```powershell
$env:SOLSCAN_TOKEN="ваш_токен"
```

**Linux / macOS:**
```bash
export SOLSCAN_TOKEN=ваш_токен
```

## Использование

### 3D-граф (GUI)

```bash
python solscan_gui.py
```

Откроется браузер на `http://127.0.0.1:8765/`. Введи один или несколько Solana-адресов,
добавь их в список и построй граф:

- **узлы** — адреса (отслеживаемые подсвечены);
- **рёбра** — агрегированные потоки `from → to` с анимацией направления;
- размер узла зависит от объёма, толщина ребра — от суммы переводов.

### CLI

```bash
# Переводы по кошельку
python solscan.py transfers So11111111111111111111111111111111111111112 -n 20

# DeFi-активность (свопы и т.п.)
python solscan.py defi <address> -n 20

# Метаданные и цена токена
python solscan.py token EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v

# Переводы по mint токена
python solscan.py ttransfers <token_mint>

# Разбор транзакции по сигнатуре
python solscan.py tx <signature>

# Мониторинг кошелька в реальном времени (+ звук, всплывашки, дозапись в CSV)
python solscan.py watch <address> --interval 30 --notify --sound --csv out.csv

# Произвольный вызов любого эндпоинта
python solscan.py raw account/transfer address=<addr> page_size=20
```

К большинству команд можно добавить `--json` (сырой JSON) или `--csv FILE` (экспорт в CSV).
